## 一、ICAO 标准陆空通话指南

### 基本原则

1. **简洁性**：用最少的话传递完整准确的信息
2. **无歧义**：避免可能产生混淆的表达
3. **标准化**：使用 ICAO 规定的短语和格式

### 关键规则

**许可的语序：** ICAO标准格式——要素先出，许可在后。如 "CSN3101, runway 01L, cleared for takeoff"，而非 "cleared for takeoff, runway 01L"。

**READ BACK 必须包含的要素：**
- 指令类型（line up, take off, descend, turn left 等）
- 所有数字参数（跑道号、高度、航向、速度、频率、应答机编码、气压值）
- 单位/限制条件（"and wait", "not above" 等）
- 本机呼号（开头或末尾）

**不需要复诵的内容：** 前瞻性信息（"expect..."）、通波信息（ATIS）、气象信息（除非被要求）。

**频率读法：** 整数可分段读（118→"one one eight"或"one eighteen"）；小数用 decimal 逐位读（118.35→"decimal three five"），禁止说 "thirty five"。

**高度读法：** 逐位读（FL120→"flight level one two zero"），非"one hundred twenty"。

**跑道号读法：** 逐位读（01L→"runway zero one left"）。

### 紧急通话格式

- **MAYDAY ×3**：遇险呼叫，格式：`MAYDAY MAYDAY MAYDAY, [呼号], [位置], [情况], [意图]`
- **PAN-PAN ×3**：紧急呼叫

## 二、民航翻译原则与方法

### 核心原则

1. 术语优先使用 CAAC 和 ICAO 官方译法
2. 缩写第一次出现时附全称（如 VOR（VHF全向信标台））
3. 保持原文技术精度，不省略"可能""在xxx条件下"等限定词
4. 陆空通话翻译保留标准句式

### 常见对应表

| 中文 | 英文 | 说明 |
|------|------|------|
| 可以起飞 | cleared for takeoff | 固定搭配，不可替换 |
| 可以落地 | cleared to land | 固定搭配 |
| 可以进入等待 | hold | 不译作 enter holding pattern |
| 联系 | contact | 不译作 call |
| 再见 | good day / good night | 不译作 bye |
| 抄收 | Roger | 仅表示收到 |
| 照办 | Wilco | 收到并执行 |
| 离地后 | after departure | 管制术语，非 after takeoff/airborne |

## 三、METAR / TAF 解码指南

### METAR 读取顺序（按决策优先级）

1. 风、能见度/RVR、天气现象 → 能否起降
2. 云（云底高、云量）→ 进近方式
3. 温度/露点、QNH → 结冰条件、高度表
4. 最近一小时变化 → 趋势判断

### RVR 与能见度对照

| 进近类型 | RVR 最低标准 |
|----------|-------------|
| CAT I | 550m |
| CAT II | 300-350m |
| CAT IIIA | 200m |
| CAT IIIB | 50-200m |
| 非精密/类精密 | 800-1200m |

着陆以跑道入口点和中间点 RVR 中较低值为准。

### TAF 关键限定词

| 词 | 含义 |
|----|------|
| FM | 确定性转变 |
| TEMPO | 短暂出现（每次<1小时） |
| BECMG | 逐步渐变 |
| PROB30/PROB40 | 概率性 |

### SIGMET 颠簸等级

| 等级 | 座舱感知 |
|------|----------|
| 轻度 LGT | 杯水面波动 |
| 中度 MOD | 杯水溅出，行走困难 |
| 严重 SEV | 人员被甩向安全带 |
| 极度 EXT | 结构受力，飞机短暂失控 |

### 结冰条件速判

温度 - 露点 ≤ 3°C 且飞机将进入云层 = 结冰风险高。

### 雷暴绕飞

- 红色回波：绕飞至少 20nm
- 两雷暴间安全间隔：≥ 40nm
- 上风方向绕飞比下风方向颠簸轻
- 气象雷达看不到 CAT 和冰晶云

## 七、关键系统操作速查

### TCAS RA 处置

| 步骤 | 动作 |
|------|------|
| 1 | 立即响应，不分析不犹豫 |
| 2 | 柔和但果断操纵，约 0.25G，1500-2000 ft/min |
| 3 | 不使用航向改变回避 |
| 4 | 机动后报告管制员 "TCAS RA" |
| 5 | 语音 "CLEAR OF CONFLICT" 后返回原高度 |

RA 优先级高于管制指令。低高度（<1100ft RA）自动抑制为 TA only。

### GPWS PULL UP 记忆项目

| 顺序 | 动作 |
|------|------|
| 1 | 断开自动驾驶和自动油门 |
| 2 | 最大推力（TOGA） |
| 3 | 抬头至 15-20°（不超过抖杆/失速保护） |
| 4 | 确认正上升率 |
| 5 | 按 FCOM 收襟翼 |
| 6 | 结合地形显示避让 |

PULL UP 与 TCAS RA 冲突时，PULL UP 优先。

### CPDLC 常用回应

| 回应 | 含义 |
|------|------|
| WILCO | 收到并承诺执行 |
| ROGER | 收到（不承诺执行） |
| STANDBY | 待处理 |
| UNABLE | 无法执行 |

### 侧风心算

| 风向夹角 | 侧风 = 风速 × |
|----------|---------------|
| 30° | 50% |
| 45° | 70% |
| 60° | 90% |

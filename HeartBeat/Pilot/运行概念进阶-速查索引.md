## 二、按飞行场景速查

### 想知道进近能不能飞？
- 📍 [航空气象解读] → RVR vs 能见度、侧风分量心算、低云决策
- 📍 [PBN导航体系] → RNP APCH / RNP AR 标准与最低能见度要求
- 📍 [GPWS与EGPWS] → 低能见度进近时 Mode 5 可能的虚假告警

### 想知道巡航阶段要注意什么？
- 📍 [RVSM与相邻空域] → 高度保持精度、高度表交叉检查
- 📍 [CPDLC数据链通信] → 数据链指令确认（WILCO）、频率变更
- 📍 [ADS-B监视技术] → 周围交通态势感知、紧急状态广播
- 📍 [TCAS与空中防撞] → TA/RA 处置、雷暴区 RA 的特殊应对
- 📍 [ETOPS延程运行] → 等时点计算、备降场天气监控、燃油策略

### 想了解特定系统的工作原理？
- **高度** → RVSM（DOC 9574 标准、高度总误差 TVE）
- **航路** → PBN（RNAV vs RNP、导航规范、ANP 监控）
- **位置** → ADS-B（Out/In、1090ES、紧急状态广播）
- **通话** → CPDLC（DCL 数字放行、AFN 登录、CRA/CLA）
- **延程** → ETOPS（等级划分、三要素认证、ETP/ERA）
- **防撞** → TCAS（TA/RA、Version 7.1、语音指令逻辑）
- **近地** → GPWS/EGPWS（7 个模式、Mode 5 下滑道偏离、PULL UP）
- **天气** → 气象（METAR 读取顺序、TAF 信任度、雷暴绕飞框架）

### 想找实操检查清单？
- 📍 [航空气象解读] → 第七节「起降阶段气象决策清单」— 8 项进近简令天气检查
- 📍 [全飞行阶段总览] → 全阶段速查表 + 每阶段飞行员关注重点

## 三、各篇关键"陷阱"提醒

| 篇目 | 容易忽略的点 |
|------|-------------|
| RVSM | 高度保持性能要求是对**机队统计**的，单次飞得好不够 |
| PBN | RNP 要求**机上性能监控告警**，RNAV 不需要——两者不能混用 |
| ADS-B | 紧急状态信息广播（7700/7600/7500）不受应答机失效影响（独立于 Mode A） |
| CPDLC | DCL 一致性检查失败必须退回语音放行，不能强行 |
| ETOPS | 发动机**空中停车率**（IFSD）是 ETOPS 等级认证的数据基础 |
| TCAS | RA 要求 **1 秒内开始响应、5 秒内完成初始机动** |
| GPWS | **PULL UP 从未因为虚假告警而被标注为"仅供参考"**——仍然需要立即执行 |
| 气象 | TAF 的 PROB30 不是"三四成概率"而是"预报员不确定但值得注意" |
